# Course

Jeu de simulation de course de voitures en C# WinForms. Compatible Windows (.NET 8) et Linux (mono).

**Auteur : Razanadrakoto Andrimalala Mitia**
Copyright (c) 2026 — Tous droits réservés. Voir [LICENSE](LICENSE).

## Lancer le projet

### Linux

```bash
sudo apt install mono-complete
./run-mono.sh
```

### Windows

```bash
dotnet run --project Course
```

Ou ouvrir `Course.sln` dans Visual Studio.

## Jouer

Maintenir **Espace** pour accélérer. Course de 3 000 m. La vitesse ne diminue pas quand on relâche.
